# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-09
### Added
- Initial public release: minimal, type-safe, text-only Rust LangChain implementation.
- LLM abstraction and implementations for OpenAI, Anthropic, Qwen, Deepseek, Ollama.
- Tool system with proc-macro for custom tool definition.
- Agent system supporting ReAct loop and tool integration.
- Example usage for Ollama, agent, and custom tools.
- TOML-based configuration system.
- Unified error handling.
- Documentation: README.md and DESIGN.md fully in English, with up-to-date examples.
